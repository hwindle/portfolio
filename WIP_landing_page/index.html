Cache

Intel Smart Cache lets all the cache share the fastest cache, and ranges from 12MB to 24MB. The cache buffers data from the CPU and RAM, and is a faster type of memory chip. Back to top

Installation

All Intel processors come with a basic cooler in the box. The CPU's do not have little gold pins on the bottom, the motherboard socket has the tiny gold pins. This is the opposite way to AMD processors. As the CPU cooler is so basic, a better cooler with a better fan and cable arrangement should be factored into the cost, and the thermal compound. Some aftermarket CPU coolers already come with a thermal pad or a tube of thermal compound. Only put a bit of thermal paste on, as otherwise it goes everywhere on the motherboard. Follow the instruction diagrams in the CPU cooler box for socket 1200, and the motherboard manual.

Frequency

Each processor has a default base frequency, which will be about 1GHZ lower, and a higher boost frequency. Keeping the CPU cool and applying a small overclock will enable the CPU to run at the boost frequency more often. Remember, overclocking requires a more expensive motherboard with a heatsink around the CPU socket area. This means the motherboard will be able to deliver more voltage to the CPU without overheating. Consider a better air cooler with a higher TDP or a liquid cooler if you intend to overclock, and choose a processor part number with a 'K' on the end.

Core Count

Due to competition from AMD Ryzen, Intel have upped their core count and thread count. For example, the 10400F has 6 cores and 12 threads and is only slightly more expensive than the AMD Ryzen 3600 (with the aftermarket CPU cooler included). However AMD Ryzen processors work with faster RAM speeds. Intel processors work with up to 2933mhz speed RAM, and AMD Ryzen works with up to 4600mhz RAM. When selecting RAM, always buy a kit with 2 sticks in it, to run the memory in dual channel mode.
